I thought that a good first post would be to talk about how I made this site.
If you look at the [github repo](https://github.com/dtjandra888/dtjandra888.github.io), this project is only consists of a few files, and something similar would be a great first project for aspiring CS students.

The project is broken down into two parts, the Jinja HTML templates (and the static JS/CSS) and the python script that renders it.
The site is automatically deployed using Github Actions and Github Pages.


## HTML
Jinja is a Python templating library that helps maintain a consistent layout across pages. This site uses a `base.html` template to share common elements like the navbar and styling.

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link rel="stylesheet" href="/static/css/bootstrap.min.css" />
    <link rel="stylesheet" href="/static/css/custom.css" />
    <link rel="shortcut icon" href="/static/img/favicon.ico" />
	<link href='https://fonts.googleapis.com/css?family=Lato:400,700' rel='stylesheet' type='text/css'>

    {% block extra_head %}{% endblock %}
    <title>{% block title %}{% endblock %}</title>
</head>
	<!-- Navigation -->
	<div class="main-nav">
		<div class="container">
			<header class="group top-nav">
			    <nav id="navbar-1" class="navbar item-navbar">
				    <ul>
				        <li><a href="/#about">About</a></li>
				        <li><a href="/#experiences">Experiences</a></li>
				        <li><a href="/#achievements">Achievements</a></li>
				        <li><a href="/#skills">Skills</a></li>
                        <li><a href="/blog.html">Blog</a></li>
				    </ul>
				</nav>
			</header>
		</div>
    </div>

    <div class="container">
        {% block content %} {% endblock %}
    </div>

    <script src="/static/js/bootstrap.min.js"></script>
</body>
</html>
```
